# Orbital Spiral

## What it is

An Archimedean spiral that can spin in place or orbit its center while leaving fading trails behind.

## Controls

- Adjust starting radius, spacing, number of turns, and resolution.
- Control spin speed, orbit radius, orbit speed, and orbit offsets.
- Toggle trails, clear the screen, or pause playback.

## Collaboration notes

- Strong extension points: color systems, path export, interaction modes, or multi-spiral compositions.
